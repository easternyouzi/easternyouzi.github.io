绪论

基本数据结构

1、线性结构：线性表；栈、队列，双队列；数组，串

2、非线性结构：树、二叉树；图，网

两类主要算法

1、查找算法：顺序、折半、分块、哈希、二叉排序树

2、排序算法：直接插入、堆排序、2-路归并，快速排序

基本概念

数据结构：相互之间存在一种或多种特定关系的数据元素的集合

数据：能输入到计算机中并被计算机程序加工处理的符号的总称

数据元素：数据的基本单位（元素、记录、结点、顶点）

数据项：数据的不可分割的最小单位

数据对象：由性质相同（类型相同）的数据元素组成的集合

数据的存储结构

1.顺序存储结构（向量、一维数组）

2.非顺序存储结构（链接表）

数据类型

一个值的集合和定义在这个值上的一组操作的总称

原子类型：如int 、char 、float

结构类型：如线性表、数组、树等

抽象数据类型

一个抽象数据类型可以用一个三元组来表示 （D、S、P）

D:数据对象

S：D中数据元素的关系集

P:对D中数据元素的基本操作

算法的时间复杂度

定义：算法（或程序）中基本操作（或语句）重复执行的次数的总和

设n为求解的问题的规模，f(n)为基本操作（或语句）执行次数总和称为语句频度。

时间复杂度算法

1、去掉f(n)中的所有加法常数。

2、只保留最高阶项

时间复杂度记作T(n), T(n)=O(f(n))

例1：

定义完变量s后有三条语句，每条语句执行1次

f(n) = f(1) = 3

时间复杂度T(n) = O(f(n)) = O(3) = O(1)

O(1)称为常量阶/常量数量级

例2：

在该函数sum中，语句频度f(n) = 1 + n + n + 1 = 2n +2

时间复杂度为：T(n) = O(f(n)) = O(2n+2) = O(n)

O(n)称为线性阶/线性数量级

例3：

在该函数sum中，语句频度f(n) = 1 + m + m*n + m*n + 1 = 2mn +2m +1

当m = n时，f(n) = 2n2+2n+1

时间复杂度为：T(n) = O(f(n)) = O(2n2+2n+1) = O(n2)

O(n)称为平方阶/平方数量阶

例4：

循环主题的执行次数为：1+2+…+n = n(1+n)/2

f(n) = 1 + n + n(n+1) + n = n2+3n+1

T(n) = O(f(n)) = O(n2)

常见的时间复杂度

常见的时间复杂度有：O(1),O(log2n),O(n),O(nlog2n),O(n2),O(n3),O(2n)
