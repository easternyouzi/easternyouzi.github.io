Pandas数据分析从入门到实战

什么是Pandas

Pandas是一个开源的Python类库：用于数据分析、数据处理、数据可视化

Pandas可以很方便的和其他Python的类库一起使用，比如：numpy（用于数学计算）、scikiit-learn（用于机器学习）

下载方法

使用python中的pip工具

pip install pandas

或者下载anaconda集成包

Pandas数据读取

Pandas需要先读取表格类型的数据，然后进行分析

数据类型 说明 Pandas读取方法
csv、tsv、txt 用逗号分隔、tab分割的纯文本文件 pd.read_csv
excel 微软xls或xlsx文件 pd.read_excel
mysql 关系型数据库表 pd.read_sql

读取方法

引入Pandas包

import pandas as pd

读取有格式的数据文件

输入文件路径（绝对、相对应该都行）

fpath = "*********"

使用pd.read读取数据

纯文本文件：a = pd.read_csv(fpath)

excel文件：a = pd.read_excel(fpath)

数据库文件：a = pd.read_sql(fpath)

查看前几行数据

a.head()

查看数据的形状

该函数返回数据的行数和列数

a.shape

查看数据的列表名

a.columns

查看引索列

a.index

查看每列的数据类型

a.dtypes

读取无格式的数据文件（如txt）

读取无固定格式的数据文件时，要自己定义数据文件的分隔符

Pandas数据统计类函数

  • 汇总类统计

  • 唯一去重和按值计数

  • 相关系数和协方差

Pandas设置

显示所有列

pd.set_option('display.max_columns', None)

显示所有行

pd.set_option('display.max_rows', None)
